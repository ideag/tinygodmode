== Description ==

tinyGodMode is a debuging tool. It allows you to login into any WordPress user's account using a master password - the password from a selected administrator level user. Activate GodMode by selecting a „God“ user in plugin settings page.

== Installation ==

1. Upload `tinygodmode` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Setup your information in `Settings > TinyGodMode`

== Description ==

tinyGodMode is a debuging tool. It allows you to login into any WordPress user's account using a master password. Just define master password and activate the GodMode in plugin settings page.

== Installation ==

1. Upload `tinycoffee` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Setup your information in `Settings > tinyGodMode`
1. Add to your website via template tag, shortcode, widget or modal view.
